# Skill Index

Skills are multi-tool workflows that fulfill user intents.

## Available Skills

| Skill | Capability | Tools Used |
|-------|------------|------------|
| [deep-research](deep-research.md) | Synthesize knowledge across documents | `catalog_search` → `chunks_search` |
| [library-discovery](library-discovery.md) | Browse and inventory content | `list_categories` → `category_search` |

## How Skills Work

1. **Read the skill file** for workflow guidance
2. **Follow the tool sequence** described
3. **Preserve context** between tool calls
4. **Produce the expected output**

## Skill Selection

Skills are selected by intents. See [intents/index.md](../intents/index.md) to match user goals.
