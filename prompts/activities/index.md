# Activity Index

Match the user's goal to an activity. Each activity may use multiple skills in sequence.

## Available Activities

| Activity | User Goal | Primary Skill | Supporting Skills |
|--------|-----------|---------------|-------------------|
| [understand-topic](understand-topic.md) | Learn about a topic | `deep-research` | `concept-exploration`, `document-analysis` |
| [know-my-library](know-my-library.md) | Discover available content | `library-discovery` | `category-exploration` |
| [explore-concept](explore-concept.md) | Track where a concept appears | `concept-exploration` | `deep-research` |
| [analyze-document](analyze-document.md) | Deep dive into a specific document | `document-analysis` | `concept-exploration` |
| [explore-category](explore-category.md) | Understand a domain's concepts | `category-exploration` | `library-discovery`, `concept-exploration` |
| [identify-patterns](identify-patterns.md) | Find applicable design patterns | `pattern-research` | `deep-research`, `document-analysis` |
| [identify-best-practices](identify-best-practices.md) | Find proven approaches | `practice-research` | `deep-research`, `concept-exploration` |
| [curate-lexicon](curate-lexicon.md) | Create a concept vocabulary for a project | `category-exploration` | `concept-exploration`, `library-discovery`, `document-analysis` |

## Quick Match

- "What does my library say about X?" → `understand-topic`
- "Research topic X" → `understand-topic`
- "What documents do I have?" → `know-my-library`
- "What categories exist?" → `know-my-library`
- "Where is [concept] discussed?" → `explore-concept`
- "Which books mention [concept]?" → `explore-concept`
- "Extract concepts from [document]" → `analyze-document`
- "What concepts are in [category]?" → `explore-category`
- "What patterns apply to X?" → `identify-patterns`
- "Best practices for X" → `identify-best-practices`
- "Create a concept lexicon for [project]" → `curate-lexicon`
- "What concepts apply to my project?" → `curate-lexicon`
