# Intent Index

Match the user's goal to an intent, then follow the linked skill.

## Available Intents

| Intent | User Goal | Skill |
|--------|-----------|-------|
| [understand-topic](understand-topic.md) | Learn about a topic | `deep-research` |
| [know-my-library](know-my-library.md) | Discover available content | `library-discovery` |
| [explore-concept](explore-concept.md) | Track where a concept appears | `concept-exploration` |
| [analyze-document](analyze-document.md) | Extract concepts from a document | `document-analysis` |
| [explore-category](explore-category.md) | Understand a domain's concepts | `category-exploration` |
| [identify-patterns](identify-patterns.md) | Identify applicable design patterns | `pattern-research` |
| [identify-best-practices](identify-best-practices.md) | Identify best practices | `practice-research` |

## Quick Match

- "What does my library say about X?" → `understand-topic`
- "Research topic X" → `understand-topic`
- "What documents do I have?" → `know-my-library`
- "What categories exist?" → `know-my-library`
- "Where is [concept] discussed?" → `explore-concept`
- "Which books mention [concept]?" → `explore-concept`
- "Extract concepts from [document]" → `analyze-document`
- "What concepts are in [category]?" → `explore-category`
- "What patterns apply to X?" → `identify-patterns`
- "Best practices for X" → `identify-best-practices`
