# Code Reference Guidelines for ADRs

**Date:** 2025-11-20  
**Principle:** ADRs are historical records - code references must reflect state at decision time

## Guideline

**Code references in ADRs should indicate the commit where the decision was implemented.**

### Format

**For implementation code:**
```
Code: src/path/file.ts (at commit abc123de)
```
